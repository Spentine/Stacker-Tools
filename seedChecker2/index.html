Seed Checker V2

The successor to Seed Checker V1. This new program uses more advanced algorithms to quickly brute force game seeds. Rather than shuffling queues, it instead only operates on numbers and more efficiently compares it to cached lists. To further multiply the speed, multithreading is implemented to make use of all logical processors on the machine, allowing for faster and more efficient seed checking on a web application. For more information, check out the README.

Configuration

Customize what the program will be searching for. The piece sequence, randomizer, and other settings can be adjusted to suit your needs.
